About Travel App

Travel App is a web application for finding places for tourism in other countries. Also Travel App provides the ability to share activity with other users of the application.

This is a project for the KRKM DNU College.

Security

Travel App is not responsible for user provided geolocation data. The user himself provides the application with the data of his location.

Travel App does not check the users of the application and is not responsible for the safety of tourists. Each tourist is advised to check the information about the landlord before the trip.
